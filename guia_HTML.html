📄 Guía Completa de HTML

1. Estructura base

Todo archivo HTML empieza con esta estructura. Sin esto, el navegador no sabe qué hacer.

<!DOCTYPE html>          <!-- le dice al navegador que es HTML5 -->
<html lang="es">         <!-- etiqueta raíz, todo va adentro -->
<head>
    <meta charset="UTF-8">    <!-- permite tildes y ñ -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <!-- viewport hace que se vea bien en celular -->
    <title>Título de la pestaña</title>
    <link rel="stylesheet" href="estilos.css">  <!-- conecta el CSS -->
</head>
<body>
    <!-- todo lo visible va aquí -->
</body>
</html>

Regla: todo lo que el usuario ve va en <body>. Todo lo de configuración va en <head>.

2. Títulos h1 - h6

Hay 6 niveles. h1 es el más grande y solo debe haber uno por página. Los demás se usan para secciones y subsecciones.

<h1>Título principal</h1>    <!-- el más grande, solo uno por página -->
<h2>Sección</h2>             <!-- subtítulo de sección -->
<h3>Subsección</h3>          <!-- subtítulo de subsección -->
<h4>Nivel 4</h4>
<h5>Nivel 5</h5>
<h6>Nivel 6 (el más pequeño)</h6>

h2 - Sección

h3 - Subsección

h5
h6

4. Enlaces <a>

Sirven para navegar a otra página, sección o sitio web.

<a href="https://google.com">Ir a Google</a>
<!-- href = la URL a donde va -->

<a href="https://google.com" target="_blank">Abrir en pestaña nueva</a>
<!-- target="_blank" = abre en pestaña nueva -->

<a href="pagina2.html">Ir a otra página del mismo sitio</a>
<!-- si el archivo está en la misma carpeta, solo pones el nombre -->

<a href="#seccion">Ir a una sección de la misma página</a>
<!-- el # apunta a un id en la misma página -->

<a href="mailto:correo@ejemplo.com">Enviar correo</a>
<!-- abre el cliente de correo -->
Abrir Google en pestaña nueva

5. Imágenes <img>

La etiqueta img no tiene cierre. src es obligatorio, alt también (buena práctica).

<img src="foto.jpg" alt="descripción">
<!-- src = ruta de la imagen -->
<!-- alt = texto si la imagen no carga (importante para accesibilidad) -->

<img src="foto.jpg" alt="foto" width="200">
<!-- width controla el ancho en píxeles -->

<img src="foto.jpg" alt="foto" width="100%">
<!-- width 100% ocupa todo el ancho disponible -->

<!-- Rutas -->
<img src="foto.jpg">               <!-- misma carpeta -->
<img src="imagenes/foto.jpg">      <!-- subcarpeta -->
<img src="https://url/foto.jpg">   <!-- imagen de internet -->

7. div y span

div = caja que ocupa toda la línea. Se usa para agrupar secciones.

span = va dentro del texto sin romper la línea. Se usa para estilizar partes.

<div>
    <!-- agrupa elementos, ocupa toda la línea -->
    <p>Párrafo dentro del div</p>
</div>

<p>Texto con <span style="color:red;">parte roja</span> en medio.</p>
<!-- span no rompe la línea -->
Esto es un div (ocupa toda la línea)

Texto con parte roja usando span.

8. class e id

class se puede repetir en muchos elementos. Se usa para aplicar el mismo estilo a varios.

id debe ser único. Se usa para identificar un elemento específico.

<!-- class: se puede repetir -->
<div class="tarjeta">Tarjeta 1</div>
<div class="tarjeta">Tarjeta 2</div>

<!-- id: único en la página -->
<h1 id="titulo-principal">Título</h1>

<!-- un elemento puede tener varias clases -->
<div class="tarjeta activa grande">...</div>

<!-- En CSS:
     .tarjeta  { }   <-- punto para class
     #titulo   { }   <-- # para id
-->

9. Formularios

Se usan para recibir datos del usuario: texto, contraseñas, selecciones, etc.

<form>
    <input type="text"     placeholder="Nombre">      <!-- campo de texto -->
    <input type="email"    placeholder="Correo">      <!-- valida formato email -->
    <input type="password" placeholder="Contraseña"> <!-- oculta lo que escribes -->
    <input type="number"   placeholder="Edad">        <!-- solo números -->
    <input type="date">                               <!-- selector de fecha -->

    <select>                                          <!-- lista desplegable -->
        <option>Opción 1</option>
        <option>Opción 2</option>
    </select>

    <textarea rows="4" placeholder="Mensaje"></textarea>  <!-- área grande de texto -->

    <label>
        <input type="checkbox"> Acepto términos        <!-- casilla de verificación -->
    </label>

    <input type="radio" name="genero" value="m"> Masculino  <!-- opción única -->
    <input type="radio" name="genero" value="f"> Femenino

    <button type="submit">Enviar</button>             <!-- botón de envío -->
    <button type="button">Botón normal</button>       <!-- botón sin acción -->
</form>

10. Tablas

Para mostrar datos en filas y columnas.

<table>
    <thead>                          <!-- cabecera de la tabla -->
        <tr>                         <!-- tr = fila -->
            <th>Nombre</th>          <!-- th = celda de cabecera (negrita) -->
            <th>Edad</th>
        </tr>
    </thead>
    <tbody>                          <!-- cuerpo de la tabla -->
        <tr>
            <td>Kevin</td>           <!-- td = celda normal -->
            <td>25</td>
        </tr>
        <tr>
            <td>Luis</td>
            <td>19</td>
        </tr>
    </tbody>
</table>

11. Etiquetas semánticas

Son como div pero con nombre descriptivo. Le dicen al navegador qué tipo de contenido hay adentro.

<header>  <!-- cabecera de la página o sección -->
    <h1>Mi sitio</h1>
    <nav>                    <!-- menú de navegación -->
        <a href="#">Inicio</a>
        <a href="#">Sobre mí</a>
    </nav>
</header>

<main>                       <!-- contenido principal -->
    <section>                <!-- sección temática -->
        <article>            <!-- contenido independiente (post, tarjeta) -->
            <h2>Título</h2>
            <p>Contenido...</p>
        </article>
    </section>

    <aside>                  <!-- contenido secundario (barra lateral) -->
        <p>Publicidad o info extra</p>
    </aside>
</main>

<footer>                     <!-- pie de página -->
    <p>© 2025 Mi sitio</p>
</footer>
