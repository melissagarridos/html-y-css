CSS — Cascading Style Sheets

El lenguaje que da estilo visual a tu HTML

¿Qué es CSS?

CSS (Hojas de Estilo en Cascada) es el lenguaje que controla cómo se ve tu HTML: colores, tamaños, fuentes, espaciado, animaciones y mucho más.

Se llama "en cascada" porque los estilos se aplican de arriba hacia abajo y los más específicos o más abajo en el archivo ganan sobre los anteriores.

📌 HTML define qué hay en la página. CSS define cómo se ve. JavaScript define cómo se comporta.

Cómo vincular CSS a HTML

Hay tres formas de agregar CSS. La mejor práctica es usar un archivo externo.

1 — Archivo externo (✅ recomendado)
<!-- Dentro del <head> de tu HTML -->
<link rel="stylesheet" href="estilos.css">
2 — Dentro del HTML con <style>
<style>
  p { color: blue; }
</style>
3 — Estilos inline (⚠️ evitar en lo posible)
<p style="color: blue; font-size: 16px;">Texto</p>
⚠️ Los estilos inline tienen la mayor especificidad y son difíciles de mantener. Úsalos solo cuando sea absolutamente necesario.

Selectores

Los selectores indican a qué elementos HTML aplicar los estilos. Son la parte más importante de CSS.

Tipos de selectores
/* Por etiqueta — aplica a TODOS los <p> */
p { color: gray; }

/* Por clase — aplica a elementos con class="tarjeta" */
.tarjeta { background: white; border-radius: 8px; }

/* Por ID — aplica al elemento con id="menu" (único) */
#menu { position: fixed; top: 0; }

/* Combinado — todos los <p> dentro de .tarjeta */
.tarjeta p { font-size: 14px; }

/* Hijo directo */
ul > li { list-style: none; }

/* Pseudo-clases — estados especiales */
a:hover         { color: purple; }         /* al pasar el mouse */
input:focus     { border-color: blue; }   /* al hacer clic */
li:first-child  { font-weight: bold; }   /* primer elemento */
li:last-child   { margin-bottom: 0; }   /* último elemento */
li:nth-child(2) { color: red; }          /* segundo elemento */

/* Pseudo-elemento */
p::first-letter { font-size: 2em; }     /* primera letra */
div::before     { content: '★'; }       /* antes del elemento */
div::after      { content: '.'; }        /* después del elemento */

/* Múltiples selectores con coma */
h1, h2, h3 { font-family: Georgia, serif; }

/* Selector universal */
* { box-sizing: border-box; }
💡 Especificidad: ID (#) > Clase (.) > Etiqueta. Cuando dos reglas aplican al mismo elemento, gana la más específica. Los estilos inline ganan sobre todos.

Box Model — El modelo de caja

Todo elemento HTML es tratado como una caja rectangular con 4 capas. Entender esto es fundamental para controlar el layout.

Propiedades del box model
div {
  width: 300px;          /* ancho del contenido */
  height: 200px;         /* alto del contenido */
  padding: 20px;         /* espacio DENTRO del borde */
  border: 2px solid #333; /* el borde visible */
  margin: 30px;          /* espacio FUERA del borde */

  /* ¡Muy importante! — incluye padding y border en width */
  box-sizing: border-box;
}

/* Shorthand — arriba | der | abajo | izq */
padding: 10px 20px 10px 20px;
padding: 10px 20px;    /* vertical | horizontal */
padding: 10px;          /* todos lados iguales */

/* Lados individuales */
padding-top: 10px;
margin-left: auto;       /* centra horizontalmente */
border-radius: 8px;     /* bordes redondeados */
border-radius: 50%;      /* círculo perfecto */
}
Visual del Box Model

🔲 border
⬛ padding
contenido (width × height)
💡 Agrega * { box-sizing: border-box; } al inicio de todo proyecto. Así el width que defines incluye el padding y border, lo que hace los cálculos mucho más intuitivos.

Propiedad display

Define cómo un elemento se comporta en el flujo del documento.

Valores de display
div    { display: block; }         /* ocupa toda la línea */
span   { display: inline; }        /* va en línea con el texto */
img    { display: inline-block; }  /* inline pero acepta width/height */
.nav  { display: flex; }          /* layout flexible (ver cap. Flexbox) */
.grid { display: grid; }          /* layout en cuadrícula */
.hide { display: none; }          /* oculta el elemento completamente */
ValorComportamiento¿Acepta width/height?
blockNueva línea, ocupa 100% del ancho✅ Sí
inlineFluye con el texto, sin salto❌ No
inline-blockFluye con texto pero acepta dimensiones✅ Sí
flexContenedor flexible (Flexbox)✅ Sí
noneEl elemento desaparece del layout—

Tipografía

Propiedades de texto
p {
  /* Fuente */
  font-family: 'Segoe UI', Arial, sans-serif; /* lista de opciones */
  font-size: 16px;    /* tamaño. También em, rem, % */
  font-weight: 400;    /* grosor: 400=normal, 700=bold */
  font-style: italic;  /* normal | italic | oblique */

  /* Texto */
  color: #333;
  text-align: center;     /* left | right | center | justify */
  text-decoration: none;  /* underline | line-through | none */
  text-transform: uppercase; /* uppercase | lowercase | capitalize */
  letter-spacing: 0.05em; /* espacio entre letras */
  line-height: 1.6;       /* altura de línea (sin unidad = relativo) */
  word-spacing: 4px;      /* espacio entre palabras */
  white-space: nowrap;    /* evita saltos de línea */

  /* Desbordamiento de texto */
  overflow: hidden;
  text-overflow: ellipsis; /* muestra "..." si no cabe */
}
💡 Usa rem para tamaños de fuente (relativo al root, predecible) y em para padding/margin (relativo al elemento padre).

Unidades y tamaños

Variables CSS (Custom Properties)

Permiten guardar valores reutilizables. Son esenciales para mantener un código consistente y fácil de mantener.

Definir y usar variables
/* Define en :root para acceso global */
:root {
  --color-primario: #534AB7;
  --color-texto: #1a1830;
  --radio: 8px;
  --sombra: 0 4px 12px rgba(0,0,0,0.1);
  --espaciado-md: 16px;
}

/* Usa con var() */
button {
  background: var(--color-primario);
  border-radius: var(--radio);
  box-shadow: var(--sombra);
  padding: var(--espaciado-md);
}

/* Valor de respaldo si la variable no existe */
color: var(--color-secundario, gray);

/* Dark mode con variables */
@media (prefers-color-scheme: dark) {
  :root {
    --color-texto: #e0ddff;
    --color-fondo: #1e1b4b;
  }
}

Posicionamiento

La propiedad position
/* static (por defecto) — flujo normal */
div { position: static; }

/* relative — se mueve desde su posición original */
div {
  position: relative;
  top: 10px;  left: 20px; /* desplazamiento */
}

/* absolute — se posiciona respecto al padre con position != static */
.icono {
  position: absolute;
  top: 10px;  right: 10px;
}

/* fixed — se fija en la ventana, no se mueve al hacer scroll */
.navbar {
  position: fixed;
  top: 0; left: 0; width: 100%;
  z-index: 1000; /* capa — mayor número = encima */
}

/* sticky — se pega al hacer scroll hasta cierto punto */
th {
  position: sticky;
  top: 0;
}
💡 Para que position: absolute funcione correctamente, el padre debe tener position: relative. Si no, el elemento se posiciona respecto al <body>.

Transiciones y animaciones

transition — cambios suaves
button {
  background: purple;
  /* propiedad | duración | función | retraso */
  transition: background 0.3s ease, transform 0.2s ease;
}
button:hover {
  background: darkpurple;
  transform: scale(1.05);
}

/* Funciones de timing */
transition-timing-function: ease;        /* suave */
transition-timing-function: linear;      /* constante */
transition-timing-function: ease-in-out; /* lento-rápido-lento */
@keyframes — animaciones complejas
/* Define la animación */
@keyframes aparecer {
  from { opacity: 0; transform: translateY(-20px); }
  to   { opacity: 1; transform: translateY(0); }
}

@keyframes pulso {
  0%   { transform: scale(1); }
  50%  { transform: scale(1.1); }
  100% { transform: scale(1); }
}

/* Aplica la animación */
.tarjeta {
  animation: aparecer 0.5s ease forwards;
  /* nombre | duración | timing | fill-mode */
}

.icono {
  animation: pulso 2s ease-in-out infinite;
  animation-delay: 0.3s; /* retraso antes de iniciar */
}

/* transform — transforma sin mover el flujo */
div {
  transform: rotate(45deg);         /* rotar */
  transform: scale(1.2);            /* agrandar */
  transform: translateX(50px);      /* mover horizontal */
  transform: translate(10px, -5px); /* x e y */
  transform: skew(10deg);           /* inclinar */
}

Diseño responsive — Media Queries

Las media queries permiten aplicar estilos distintos según el tamaño de la pantalla.

Media queries
/* Mobile first: diseña para móvil primero */
.contenedor {
  padding: 1rem;
  flex-direction: column; /* columna en móvil */
}

/* Tablet y mayor (768px+) */
@media (min-width: 768px) {
  .contenedor {
    padding: 2rem;
    flex-direction: row; /* fila en tablet */
  }
}

/* Desktop (1024px+) */
@media (min-width: 1024px) {
  .contenedor { max-width: 1200px; margin: 0 auto; }
}

/* Orientación */
@media (orientation: landscape) { ... }

/* Dark mode del sistema */
@media (prefers-color-scheme: dark) { ... }

/* Impresión */
@media print { .sidebar { display: none; } }

Flexbox

El sistema de layout más potente para alinear y distribuir elementos

¿Qué es Flexbox?

Flexbox (Flexible Box Layout) es un sistema de layout de CSS que distribuye elementos hijos en una fila o columna. Se activa poniendo display: flex en el contenedor padre.

Tiene dos ejes:

  • Main axis (eje principal): horizontal por defecto (→)
  • Cross axis (eje transversal): perpendicular al principal (↓)

Propiedades del contenedor (padre)

Todas las propiedades del contenedor flex
.contenedor {
  display: flex;            /* activa flexbox */

  /* Dirección del eje principal */
  flex-direction: row;            /* → izquierda a derecha (defecto) */
  flex-direction: row-reverse;   /* ← derecha a izquierda */
  flex-direction: column;         /* ↓ arriba a abajo */
  flex-direction: column-reverse; /* ↑ abajo a arriba */

  /* Alineación en el eje principal (horizontal por defecto) */
  justify-content: flex-start;    /* al inicio (defecto) */
  justify-content: flex-end;      /* al final */
  justify-content: center;        /* al centro */
  justify-content: space-between; /* espacio entre elementos */
  justify-content: space-around;  /* espacio alrededor */
  justify-content: space-evenly;  /* espacio perfectamente igual */

  /* Alineación en el eje transversal (vertical por defecto) */
  align-items: stretch;           /* estira (defecto) */
  align-items: flex-start;        /* alinea arriba */
  align-items: center;            /* centra verticalmente */
  align-items: flex-end;          /* alinea abajo */
  align-items: baseline;          /* alinea por línea base del texto */

  /* Saltos de línea */
  flex-wrap: nowrap;              /* no salta (defecto) */
  flex-wrap: wrap;                /* salta a siguiente línea */
  flex-wrap: wrap-reverse;        /* salta en sentido inverso */

  /* Espacio entre hijos */
  gap: 16px;                      /* igual en ambos ejes */
  gap: 10px 20px;                 /* row-gap | column-gap */

  /* Shorthand: direction | wrap */
  flex-flow: row wrap;
}

Propiedades de los hijos

Propiedades de los items flex
.hijo {
  /* flex-grow: cuánto puede crecer respecto a los otros */
  flex-grow: 0;        /* no crece (defecto) */
  flex-grow: 1;        /* crece para llenar espacio disponible */
  flex-grow: 2;        /* crece el doble que flex-grow: 1 */

  /* flex-shrink: cuánto puede encoger */
  flex-shrink: 1;      /* puede encoger (defecto) */
  flex-shrink: 0;      /* NO encoge nunca */

  /* flex-basis: tamaño inicial antes de crecer/encoger */
  flex-basis: auto;    /* usa width/height (defecto) */
  flex-basis: 200px;   /* tamaño base de 200px */
  flex-basis: 33%;     /* porcentaje del contenedor */

  /* Shorthand: grow | shrink | basis */
  flex: 1;             /* = flex: 1 1 0 (crece y encoge igual) */
  flex: 0 0 200px;    /* tamaño fijo de 200px */
  flex: none;          /* = 0 0 auto (no crece ni encoge) */

  /* Alineación individual — sobreescribe align-items del padre */
  align-self: center;
  align-self: flex-end;

  /* Orden visual (sin cambiar el HTML) */
  order: 0;            /* valor por defecto */
  order: -1;           /* va primero */
  order: 2;            /* va al final */
}

Centrar elementos — el truco clásico

Centrar perfectamente en pantalla
.pantalla {
  display: flex;
  justify-content: center;  /* centra horizontalmente */
  align-items: center;      /* centra verticalmente */
  height: 100vh;             /* toda la altura de la ventana */
  min-height: 100vh;
}

HTML — HyperText Markup Language

La estructura y el contenido de toda página web

Estructura base de un documento

Plantilla HTML completa
<!DOCTYPE html>                <!-- Le dice al navegador qué versión de HTML -->
<html lang="es">               <!-- idioma del documento -->

<head>                           <!-- metadatos, no visible en pantalla -->
  <meta charset="UTF-8">        <!-- codificación de caracteres -->
  <meta name="viewport"
        content="width=device-width, initial-scale=1.0">  <!-- responsive -->
  <meta name="description" content="Descripción para SEO">
  <title>Título de la pestaña</title>
  <link rel="stylesheet" href="estilos.css">
  <link rel="icon" href="favicon.ico">  <!-- icono de pestaña -->
</head>

<body>                           <!-- todo lo visible va aquí -->
  <h1>Hola mundo</h1>
  <p>Primer párrafo</p>

  <!-- El script al final para no bloquear la carga -->
  <script src="app.js"></script>
</body>
</html>

JavaScript con CSS y HTML

Cómo conectar JS para manipular estilos y el DOM dinámicamente

Seleccionar elementos del DOM

Antes de modificar cualquier elemento, necesitas seleccionarlo. Estas son las formas principales:

Métodos de selección
// Por ID — devuelve un solo elemento
const boton = document.getElementById('mi-boton');

// Por selector CSS — devuelve el PRIMERO que coincide
const tarjeta  = document.querySelector('.tarjeta');
const titulo   = document.querySelector('h1');
const activo   = document.querySelector('#menu .activo');

// Por selector CSS — devuelve TODOS los que coinciden (NodeList)
const botones  = document.querySelectorAll('.btn');
const inputs   = document.querySelectorAll('input[type="text"]');

// Iterar sobre todos
botones.forEach(btn => {
  btn.style.color = 'red';
});

// Seleccionar dentro de otro elemento
const menu   = document.getElementById('menu');
const primer = menu.querySelector('li'); // solo busca dentro de #menu

// Navegar por el DOM
elemento.parentElement         // elemento padre
elemento.children              // hijos directos
elemento.nextElementSibling    // siguiente hermano
elemento.previousElementSibling // hermano anterior
elemento.firstElementChild     // primer hijo

Cambiar estilos con JavaScript

Modificar estilos inline desde JS
const caja = document.querySelector('.caja');

// Cambiar propiedades CSS (camelCase en JS, no kebab-case)
caja.style.color = 'red';
caja.style.backgroundColor = '#f0f0f0';  // background-color → backgroundColor
caja.style.fontSize = '20px';             // font-size → fontSize
caja.style.borderRadius = '8px';
caja.style.display = 'none';             // ocultar
caja.style.display = 'block';            // mostrar
caja.style.transform = 'rotate(45deg)';

// Leer estilos computados (los que realmente aplican)
const estilosActuales = getComputedStyle(caja);
const colorActual = estilosActuales.getPropertyValue('color');
const tamActual   = estilosActuales.getPropertyValue('font-size');
⚠️ Cambiar style.propiedad agrega estilos inline, lo que tiene alta especificidad y es difícil de mantener. La mejor práctica es usar clases CSS.

classList — la forma correcta de cambiar estilos

Métodos de classList
const el = document.querySelector('.caja');

// Agregar clase
el.classList.add('activo');
el.classList.add('grande', 'azul');     // múltiples a la vez

// Quitar clase
el.classList.remove('activo');

// Toggle — agrega si no tiene, quita si tiene
el.classList.toggle('activo');           // ideal para botones on/off
el.classList.toggle('oscuro', condicion); // agrega si condicion es true

// Verificar si tiene una clase
const tieneClase = el.classList.contains('activo'); // true/false

// Reemplazar una clase por otra
el.classList.replace('rojo', 'verde');

Tipos de eventos y cómo usarlos

  if (e.key === 'Enter') enviar();

input.addEventListener('keyup',     e => { ... }); // al soltar

// FORMULARIO
input.addEventListener('input', e => {
  console.log(e.target.value);  // valor actual mientras escribe
});
input.addEventListener('change', e => { ... }); // al perder foco con cambio
input.addEventListener('focus',  e => { ... }); // al hacer clic en el input

Variables CSS desde JavaScript

Leer y escribir variables CSS con JS
// ── LEER una variable CSS ──
const root   = document.documentElement;
const estilos = getComputedStyle(root);
const color = estilos.getPropertyValue('--color-primario').trim();
console.log(color); // "#534AB7"

// ── CAMBIAR una variable CSS ──
// Esto afecta TODA la página porque se cambia en :root
root.style.setProperty('--color-primario', '#FF6B6B');
root.style.setProperty('--fondo', '#1a1a2e');

// ── Ejemplo real: Dark Mode ──
function toggleDarkMode() {
  const isDark = document.body.classList.toggle('dark');

  root.style.setProperty(
    '--bg',
    isDark ? '#1a1a2e' : '#ffffff'
  );
  root.style.setProperty(
    '--texto',
    isDark ? '#e0ddff' : '#1a1830'
  );

  // Guardar preferencia
  localStorage.setItem('tema', isDark ? 'dark' : 'light');
}

// ── Ejemplo: Tema dinámico con slider ──
const slider = document.getElementById('hue-slider');
slider.addEventListener('input', e => {
  const hue = e.target.value;
  root.style.setProperty('--color-primario', `hsl(${hue}, 70%, 50%)`);
});
Patrón completo: app con tema dinámico
// CSS:
:root {
  --hue: 244;
  --color-primario: hsl(var(--hue), 70%, 55%);
  --color-claro:    hsl(var(--hue), 70%, 95%);
}

// JS:
document.getElementById('slider').addEventListener('input', e => {
  // Cambiar solo el hue actualiza automáticamente TODOS los colores
  document.documentElement.style.setProperty('--hue', e.target.value);
});
