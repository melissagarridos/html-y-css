🎨 Guía Completa de CSS

1. Cómo conectar CSS al HTML

Hay 3 formas. La mejor es un archivo separado.

<!-- FORMA 1: Archivo separado (recomendada) -->
<link rel="stylesheet" href="estilos.css">
<!-- va en el <head> del HTML -->

<!-- FORMA 2: Dentro del HTML con <style> -->
<style>
    p { color: red; }
</style>

<!-- FORMA 3: Directo en el elemento (evitar, difícil de mantener) -->
<p style="color: red;">Texto rojo</p>

2. Selectores - a quién le aplica el estilo

/* Por etiqueta: aplica a TODOS los <p> */
p { color: gray; }

/* Por clase: aplica a todo lo que tenga class="tarjeta" */
.tarjeta { background: blue; }

/* Por id: aplica solo al elemento con id="cabecera" */
#cabecera { font-size: 32px; }

/* Varios a la vez */
h1, h2, h3 { color: navy; }

/* Hijo directo: solo los <p> que están dentro de .contenedor */
.contenedor > p { color: red; }

/* Descendiente: todos los <p> dentro de .contenedor (a cualquier nivel) */
.contenedor p { color: red; }

/* Al pasar el mouse */
.boton:hover { background: darkblue; }

/* Al hacer click */
.boton:active { background: black; }

/* Cuando un input está seleccionado */
input:focus { border-color: blue; }

5. Margin y Padding

margin = espacio AFUERA del elemento (lo aleja de otros elementos)

padding = espacio ADENTRO del elemento (aleja el contenido del borde)

/* Todos los lados iguales */
margin: 20px;
padding: 20px;

/* Arriba/abajo  Izquierda/derecha */
margin: 10px 20px;

/* Arriba  Derecha  Abajo  Izquierda (sentido horario) */
margin: 10px 20px 5px 15px;

/* Lado específico */
margin-top: 10px;
margin-right: 20px;
margin-bottom: 10px;
margin-left: 20px;

/* Centrar un elemento horizontalmente */
margin: 0 auto;
margin:20px (espacio afuera) — padding:5px (espacio adentro)
margin:5px — padding:30px (mucho espacio adentro)

7. Width y Height - tamaño

8. Display - cómo se comporta el elemento

display: block;    /* ocupa toda la línea (div, p, h1 son block por defecto) */
display: inline;   /* va en línea con el texto (span, a son inline por defecto) */
display: inline-block; /* en línea pero acepta width y height */
display: none;     /* oculta el elemento completamente */
display: flex;     /* activa flexbox (organiza hijos en fila) */
display: grid;     /* activa grid (organiza en cuadrícula) */

9. Flexbox - organizar en fila o columna

Se activa en el contenedor padre con display: flex. Los hijos se organizan automáticamente.

.contenedor {
    display: flex;                   /* activa flexbox */
    flex-direction: row;             /* fila (por defecto) */
    flex-direction: column;          /* columna */
    justify-content: flex-start;     /* alinea al inicio (por defecto) */
    justify-content: center;         /* centra horizontalmente */
    justify-content: flex-end;       /* alinea al final */
    justify-content: space-between;  /* espacio entre elementos */
    justify-content: space-around;   /* espacio alrededor de elementos */
    align-items: stretch;            /* estira los hijos (por defecto) */
    align-items: center;             /* centra verticalmente */
    align-items: flex-start;         /* alinea arriba */
    gap: 20px;                       /* espacio entre elementos */
    flex-wrap: wrap;                 /* si no caben, bajan a la siguiente línea */
}

10. Grid - organizar en cuadrícula

.contenedor {
    display: grid;
    grid-template-columns: repeat(3, 1fr); /* 3 columnas iguales */
    grid-template-columns: 200px 1fr 1fr;  /* primera fija, las otras iguales */
    grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); /* responsive automático */
    gap: 15px;                             /* espacio entre celdas */
    grid-template-rows: 100px auto;        /* altura de filas */
}

11. Position - mover elementos libremente

position: static;    /* por defecto, flujo normal */

position: relative;  /* se mueve RELATIVO a donde estaba */
top: 10px;           /* baja 10px desde su posición original */
left: 20px;          /* se mueve 20px a la derecha */

position: absolute;  /* se mueve RELATIVO al padre con position:relative */
top: 0;              /* pegado arriba del padre */
right: 0;            /* pegado a la derecha del padre */

position: fixed;     /* se queda fijo en pantalla aunque hagas scroll */
top: 0;              /* pegado arriba de la pantalla (navbar fijo) */

position: sticky;    /* normal hasta que llegas a él, luego se queda fijo */

14. Animaciones con @keyframes

@keyframes nombre {
    from { opacity: 0; }          /* estado inicial */
    to   { opacity: 1; }          /* estado final */
}

@keyframes nombre {
    0%   { transform: scale(1); }
    50%  { transform: scale(1.5); }
    100% { transform: scale(1); }
}

.elemento {
    animation: nombre duración tipo repetición;
    animation: pulsar 1s ease infinite;    /* infinito */
    animation: aparecer 0.5s ease forwards; /* forwards = se queda en el estado final */

15. Responsive - adaptarse al tamaño de pantalla

/* Estilos base (pantalla grande) */
.contenedor {
    display: flex;
    gap: 20px;
}

/* Tablet (menor a 768px) */
@media (max-width: 768px) {
    .contenedor {
        flex-direction: column;
    }
}

/* Celular (menor a 480px) */
@media (max-width: 480px) {
    .contenedor {
        padding: 10px;
    }
    h1 {
        font-size: 20px;
    }
}

/* Tamaños de referencia */
/* max-width: 480px  -> celular pequeño  */
/* max-width: 768px  -> tablet           */
/* max-width: 1024px -> laptop           */

16. Variables CSS - reutilizar valores

/* Se definen en :root para que estén disponibles en toda la página */
:root {
    --color-principal: #007bff;
    --color-secundario: #2ecc71;
    --fuente-grande: 24px;
    --radio: 10px;
    --sombra: 0 4px 15px rgba(0,0,0,0.15);
}

/* Se usan con var() */
.boton {
    background: var(--color-principal);
    border-radius: var(--radio);
    box-shadow: var(--sombra);
}

/* Ventaja: cambias el valor en :root y cambia en todos lados */

17. Pseudoclases - seleccionar elementos específicos

li:first-child      { color: red; }     /* primer elemento */
li:last-child       { color: green; }   /* último elemento */
li:nth-child(2)     { color: blue; }    /* el segundo exacto */
li:nth-child(odd)   { background: #eee; } /* impares: 1,3,5... */
li:nth-child(even)  { background: #fff; } /* pares: 2,4,6... */

a:visited           { color: purple; }  /* enlace ya visitado */
input:disabled      { opacity: 0.5; }   /* input desactivado */
input:checked       { }                 /* checkbox marcado */
  • Primero (rojo)
  • Segundo (azul)
  • Tercero (fondo azul claro)
  • Último (verde)

18. Overflow - controlar contenido que se sale

overflow: visible;  /* por defecto, el contenido se sale */
overflow: hidden;   /* oculta lo que se sale */
overflow: scroll;   /* siempre muestra barra de scroll */
overflow: auto;     /* scroll solo si es necesario (el más usado) */

overflow-x: hidden; /* solo controla el horizontal */
overflow-y: auto;   /* solo controla el vertical */
Este texto es muy largo y se sale del contenedor, pero overflow:auto agrega scroll automáticamente para que puedas verlo todo.
